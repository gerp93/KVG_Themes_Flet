## Color Usage in Flet Material 3

Each theme maps colors to Flet's Material 3 ColorScheme properties:

- **primary**: Main brand color, used for prominent UI elements
- **on_primary**: Text/icons on primary colored surfaces
- **secondary**: Supporting color for accents
- **surface**: Background for cards, sheets, menus
- **background**: Page/app background
- **error**: Error states and warnings
- **outline**: Borders and dividers

All themes use `use_material3=True` for modern Material Design 3 components.

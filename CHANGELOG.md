## [1.0.0] - 2024-12-07

### Added
- Initial release of KVG Themes for Flet
- Ported 9 themes from KVG_Themes TKinter library:
  - Light Theme: Clean professional look with grays and blues
  - Dark Theme: Comfortable dark mode with good contrast
  - Neon Theme: Electric colors on black background
  - Retrowave Theme: 80s synthwave sunset vibes
  - Hacker Theme: Matrix-style green on black
  - Lava Theme: Fiery reds, oranges, and yellows
  - Electric Lime Theme: Bright yellow-green with purple accents
  - Bubblegum Theme: Hot pink with candy colors
  - Commander Keen Theme: Classic DOS EGA/VGA palette
- Core theme management system with registry
- Theme selector API (get_theme, get_theme_list, theme_exists)
- Complete example application with live theme switcher
- Full documentation with usage examples
- Type hints for all public APIs
- Material 3 design integration

### Features
- Zero dependencies beyond Flet
- Plug-and-play theme application
- Easy-to-use API
- Extensible theme registry
- Complete color scheme definitions for each theme
